# Catalog Fixture

Catalog Fixture is a small procedural smoke level kept in the benchmark domain to exercise descriptor discovery without changing a central registry.

## Visual language
It is intentionally minimal and uses no external assets.

## Musical language
The fixture exposes the same level contract while leaving musical detail to real benchmark outputs.

## Mechanical signature
Its runtime is deliberately lightweight because this directory exists to test catalog wiring.

## What to read
- `src/benchmark-levels/catalog-fixture/level.json`
- `src/benchmark-levels/catalog-fixture/index.ts`

## Status & notes
This is a discovery fixture, not a curated reference level.
