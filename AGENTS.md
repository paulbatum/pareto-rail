# AGENTS.md

## Project notes

- This is a Vite + TypeScript browser game using three.js WebGPU rendering only.
- Keep gameplay logic under `src/game/` and keep mesh/material/effect creation under `src/visuals/`.
- Placeholder visuals should remain deliberately simple so a later visual pass can replace them without changing game logic.
- Use `npm run typecheck` and `npm run build` before handing work back.
