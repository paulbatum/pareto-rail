# Prompt exemplars

These are ineligible prompts used to calibrate theme detail and test the runner before the benchmark freeze. They must never be selected for an eligible run or included in benchmark analysis. `controller-smoke-assignment.md` and `prism-warm-palette.md` define the bounded real-model controller smoke test; its output is always rehearsal-only.

Use a rehearsal theme that does not share its title or central concept with a hand-built playable level visible in the entrant worktree. Registry entries marked `technical` are excluded from benchmark overlap, distinctiveness, and quality-reference checks. Using an exemplar in rehearsal does not make the resulting level an entrant. Record rehearsal status explicitly in its private run manifest.
