# Prompt exemplars

These are ineligible prompts used to calibrate theme detail and test the runner before the benchmark freeze. They must never be selected for an eligible run or included in the three-theme analysis.

Use a rehearsal theme that does not share its title or central concept with a hand-built level visible in the entrant worktree. Using an exemplar in rehearsal does not make the resulting level an entrant. Record rehearsal status explicitly in its private run manifest.
